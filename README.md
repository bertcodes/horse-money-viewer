# 马钱行情查看器

一个简洁实用的股票行情查看工具，支持A股、港股、美股实时行情。

## 功能特性

- **实时行情**: 支持 A股、港股、美股实时行情数据
- **成本管理**: 记录股票成本价和持仓数量，自动计算收益率和收益
- **目标止损**: 设置目标价和止损价，实时显示价格状态
- **盈亏比**: 自动计算风险收益比，辅助投资决策
- **图表展示**: 支持分时图、日K线、周K线、月K线
- **隐蔽模式**: 快速切换隐蔽模式，保护隐私
- **自动刷新**: 可配置的自动刷新功能
- **数据导入导出**: 支持股票配置的导入和导出
- **列自定义**: 可自定义显示的列
- **排序功能**: 支持按任意列排序
- **响应式设计**: 支持移动端访问

## 使用方法

### 添加股票

在输入框中输入股票代码，支持以下格式：
- A股: `sh600519`, `sz000001`
- 港股: `hk00700`
- 美股: `usAAPL`

也可以一次性输入成本价和持仓数量，格式为：`代码,成本价,持仓数量`
例如：`sh600519,1000,100`

### 编辑股票

点击表格中的成本价、持仓、目标价或止损价单元格，即可弹出编辑窗口。

### 查看图表

- 双击股票行打开图表窗口
- 右键点击股票行选择图表类型
- 支持分时图、日K线、周K线、月K线

### 快捷键

- `Ctrl + H`: 切换隐蔽模式（老板键）
- `Ctrl + R`: 刷新数据
- `ESC`: 关闭弹窗
- `Enter`: 在编辑窗口中保存

### 隐蔽模式

点击"隐蔽"按钮或按 `Ctrl + H` 进入隐蔽模式，页面将显示为普通文档样式，保护隐私。

## 技术栈

- 纯 HTML/CSS/JavaScript
- 使用 localStorage 存储数据
- 使用腾讯财经 API 获取实时数据

## 浏览器支持

- Chrome/Edge (推荐)
- Firefox
- Safari
- 其他现代浏览器

## 注意事项

- 数据来源于第三方 API，可能存在延迟
- 指数不支持图表显示
- 停牌股票可能无法获取最新数据
- 建议在非工作时间使用，以免影响工作
